# D Music (Spotify Clone) 🎵

D Music is a feature-rich music streaming application inspired by Spotify. Built with Jetpack Compose, this project focuses on delivering a high-fidelity user interface and a smooth audio experience.

---

## 🎧 Features
- **Modern Player UI:** Sleek playback controls with album art integration.
- **Search System:** Effortlessly find tracks and artists (Work in Progress).
- **Jetpack Compose:** Fully declarative UI for maximum performance.
- **Material 3 Design:** Following the latest Android design standards.

## 🛠 Tech Stack
- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Media:** Media3 / ExoPlayer
- **Architecture:** MVVM
- **Images:** Coil
