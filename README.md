D Music 🎧
A Premium Spotify-Inspired Music Streaming Experience
D Music is a high-performance Android application built with Kotlin and Jetpack Compose. It features a modern, fluid user interface designed to provide a seamless music streaming experience, following the latest Material 3 design standards.

📥 Download & Install
Try out the latest build directly on your Android device:

Download D Music APK v1.0.0

Note: As this is a direct APK installation, you may need to allow "Install from Unknown Sources" in your device settings.

✨ Key Features
Modern Playback UI: A beautiful, animated player with album art integration and playback controls.

Dynamic Search: (In Progress) Real-time filtering to find your favorite tracks and artists.

Custom Playlists: (Coming Soon) Create and manage your own music collections.

High-Fidelity Audio: Powered by Media3 / ExoPlayer for low-latency, high-quality streaming.

Image Optimization: Utilizing Coil for smooth, memory-efficient loading of album covers.

🛠 Tech Stack
Language: Kotlin

UI Framework: Jetpack Compose (100% Declarative UI)

Media Engine: Android Media3 & ExoPlayer

Architecture: MVVM (Model-View-ViewModel)

Dependency Injection: Hilt / Koin

Image Loading: Coil
